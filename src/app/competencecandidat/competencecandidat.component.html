OPTEAMIS

TanitJobs


🔝Points forts :

  • Navigation intuitive : Filtres clairs pour une recherche efficace.
  • Alertes personnalisées : Notifications pour les nouvelles offres.
  • Adaptation locale : Conçue pour le marché tunisien, particulièrement dans le domaine IT .


🔻Points faibles :

  • Design obsolète : Interface peu moderne et complexe pour les nouveaux utilisateurs.
  • Fonctionnalités limitées : Importation de CV au format PDF peu fiable.
  • Manque d'outils recruteurs : Absence de fonctionnalité pour l'affectation directe des candidats .

OPTEAMIS


🔝 Points forts :

  • Automatisation efficace : Simplifie les processus complexes pour améliorer la productivité .
  • Base de données riche : Contient des offres variées, locales et internationales .


🔻Points faibles :

  • Fonctionnalités limités : Absence d'importation de CV en PDF, scraping de données, et outils pour affecter un candidat à une mission.
  • Accessibilité et design : Expérience utilisateur nécessitant une modernisation .
  • Recherche imprécise : Résultats parfois non pertinents dans les filtres .
  • Manque de gestion : Absence de sections pour réclamations, suggestions et retours utilisateurs. .
